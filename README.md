# RIS-v1

## What is RIS?

RIS (Resume Intelligence System) is an upgraded ATS.

Traditional ATS checks exact keyword matches.

RIS checks:
- Exact skill matches
- Transferable skills

Example:

Java → Python
MySQL → PostgreSQL
React → Vue

## Scores

1. ATS Score
   - Exact keyword matching

2. RIS Score
   - Exact matching + transferable skills
